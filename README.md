# 用户参数

要配置您的 `user_parameters.csv`，请按照以下说明操作：

1. 打开 `user_parameters.csv` 文件。
2. 为每一行填入以下信息：
    - **私钥（Private Key）**：您的私钥。
    - **猫的ID（Cat ID）**：您猫的ID。
    - **喂食次数（Feed）**：填入 `7` 表示每天喂食7次。
    - **清理次数（Clean）**：填入 `2` 表示每天清理2次。

每一行代表一个猫的任务序列。任务按顺序执行，每个任务之间有1分钟的冷却时间。

根据您的需求，可以调整冷却时间或其他参数。
